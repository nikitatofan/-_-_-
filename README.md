# Проект: Обучение с учителем - Прогнозирование снижения покупательской активности

## Постановка задачи
Разработать решение для персонализации предложений постоянным клиентам с целью увеличения их покупательской активности.

## Цель
Создать действенный механизм удержания ценных клиентов через персонализированный подход.

## Данные
- market_file.csv - основные данные о клиентах
- market_money.csv - данные о выручке по периодам
- market_time.csv - данные о времени на сайте
- money.csv - данные о прибыли

## Используемые модели
1. DecisionTreeClassifier (лучшая модель)
   - max_depth=5
   - min_samples_split=4
   - ROC-AUC: 0.869

2. LogisticRegression
3. KNeighborsClassifier
4. SVC

## Результаты
- Лучшая модель: DecisionTreeClassifier
- ROC-AUC на тестовой выборке: 0.869
- Выявлены ключевые факторы снижения активности:
  - Время на сайте
  - История выручки
  - Количество просмотренных страниц

## Сегментация клиентов
Выделен сегмент "Высокий риск - Высокая прибыльность" (12 клиентов):
- Вероятность снижения активности: 100%
- Средняя прибыль: 5.26
- Рекомендованы персональные предложения
